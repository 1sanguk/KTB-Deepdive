# 인공지능과 머신러닝

## 인공지능이란?

인공지능(AI, Artificial Intelligence)은 인간의 지능적 행동을 컴퓨터로 모방하는 기술이다. 학습, 추론, 문제 해결, 언어 이해, 시각 인식 등의 기능을 포함한다. 1956년 다트머스 회의에서 존 매카시가 처음 'Artificial Intelligence'라는 용어를 사용하면서 분야가 공식적으로 시작됐다.

## 머신러닝이란?

머신러닝(Machine Learning)은 AI의 한 분야로, 데이터로부터 스스로 패턴을 학습하는 알고리즘이다. 명시적인 규칙을 프로그래밍하는 대신, 대량의 데이터를 보여주면 모델이 자동으로 규칙을 발견한다. 크게 지도 학습, 비지도 학습, 강화 학습으로 나뉜다.

**지도 학습(Supervised Learning)**: 정답이 있는 데이터로 학습. 예) 이메일이 스팸인지 분류, 집값 예측.

**비지도 학습(Unsupervised Learning)**: 정답 없이 데이터의 구조를 스스로 발견. 예) 고객 군집화, 이상치 탐지.

**강화 학습(Reinforcement Learning)**: 환경과 상호작용하며 보상을 최대화하는 전략을 학습. 예) 게임 AI, 로봇 제어.

## 딥러닝이란?

딥러닝(Deep Learning)은 머신러닝의 한 분야로, 인간 뇌의 신경망을 모방한 인공 신경망(ANN)을 여러 층으로 쌓아 학습한다. '딥(Deep)'은 신경망의 층이 깊다는 의미다. 이미지 인식, 자연어 처리, 음성 인식에서 기존 방법보다 월등히 높은 성능을 보인다.

## 주요 딥러닝 아키텍처

**CNN (합성곱 신경망)**: 이미지 분류와 객체 인식에 특화. 픽셀의 공간적 관계를 학습한다. 대표 모델: ResNet, VGG, EfficientNet.

**RNN (순환 신경망)**: 순서가 있는 데이터(시계열, 텍스트)를 처리. 이전 상태를 기억해 다음 출력에 반영한다. 단점: 긴 시퀀스에서 기울기 소실 문제.

**LSTM / GRU**: RNN의 기울기 소실 문제를 게이트 메커니즘으로 해결. 장기 의존성을 더 잘 학습한다.

**Transformer**: 2017년 "Attention Is All You Need" 논문에서 제안. Self-Attention 메커니즘으로 시퀀스의 전체 맥락을 한 번에 파악한다. 현재 NLP의 표준 아키텍처. GPT, BERT, T5 등이 Transformer 기반이다.

## 자연어 처리(NLP)

자연어 처리(NLP, Natural Language Processing)는 컴퓨터가 인간의 언어를 이해하고 생성하는 기술이다. 번역, 요약, 질의응답, 감정 분석, 챗봇 등이 NLP 응용 분야다.

**토크나이저(Tokenizer)**: 텍스트를 모델이 처리할 수 있는 작은 단위(토큰)로 분할한다. BPE(Byte Pair Encoding), WordPiece, SentencePiece 등의 방식이 있다.

**임베딩(Embedding)**: 단어나 문장을 숫자 벡터로 변환. Word2Vec, GloVe, FastText 등이 있으며, Transformer 모델은 문맥에 따라 다른 임베딩을 생성한다.

**사전 학습 + 파인튜닝**: 대량의 텍스트로 언어 모델을 먼저 학습(사전 학습)하고, 특정 태스크 데이터로 추가 학습(파인튜닝)하는 방식이 현재 NLP의 주류다.

## 대형 언어 모델(LLM)

LLM(Large Language Model)은 수억~수천억 개의 파라미터를 가진 대규모 언어 모델이다. GPT-4, Claude, Gemini, LLaMA 등이 대표적이다. 방대한 텍스트 데이터를 학습해 자연스러운 문장 생성, 코드 작성, 번역, 요약 등 다양한 태스크를 수행할 수 있다.

**파라미터(Parameter)**: 모델이 학습을 통해 얻은 가중치 값. 파라미터가 많을수록 더 복잡한 패턴을 학습할 수 있지만, 계산 비용도 증가한다.

**프롬프트 엔지니어링**: LLM에서 원하는 출력을 얻기 위해 입력(프롬프트)을 최적화하는 기술. Few-shot, Chain-of-Thought 등의 기법이 있다.

**RAG (Retrieval-Augmented Generation)**: LLM이 외부 지식 베이스에서 관련 문서를 검색해 답변을 생성하는 기법. 모델의 학습 데이터에 없는 최신 정보나 특화 지식을 활용할 수 있다.

## AI의 학습 과정

AI 모델 학습의 핵심 개념:

- **손실 함수(Loss Function)**: 모델 예측과 정답의 차이를 수치화. 이 값을 최소화하는 방향으로 학습한다.
- **역전파(Backpropagation)**: 손실 함수의 기울기를 계산해 각 파라미터를 조정하는 알고리즘.
- **최적화 알고리즘**: SGD, Adam, AdamW 등. Adam이 가장 널리 쓰인다.
- **과적합(Overfitting)**: 학습 데이터에만 지나치게 맞춰져 새로운 데이터에 성능이 떨어지는 현상. 드롭아웃, 정규화, 데이터 증강으로 방지한다.
- **조기 종료(Early Stopping)**: 검증 손실이 더 이상 감소하지 않으면 학습을 멈춰 과적합을 방지한다.

## AI 윤리와 한계

AI는 학습 데이터에 포함된 편향(Bias)을 학습할 수 있어 공정성 문제가 발생할 수 있다. 또한 블랙박스 특성으로 결정 과정을 설명하기 어렵다(설명 가능성 문제). 환각(Hallucination) 현상 — 사실이 아닌 내용을 그럴듯하게 생성하는 것 — 도 LLM의 주요 한계 중 하나다.
